# Project: News Genie

An AI-powered news and information assistant, built with Next.js, FastAPI,
LangGraph, the Anthropic Claude API, and the NewsAPI. This one's public — it's one
of two repos on her GitHub that visitors can actually go read the code for, rather
than taking a description on faith.

It's also worth noting this project runs on Anthropic's Claude API, distinct from
the OpenAI stack behind most of her other projects (including this assistant you're
talking to right now). She's not locked into a single LLM provider — the choice
follows the project, not the other way around.
